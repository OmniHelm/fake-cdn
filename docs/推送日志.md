## /融合 CDN/开放API/推送日志
```text
 1.一条数据, 一次请求, 不支持批处理以及符号隔开操作
 
 2.注意请求的数据类型, 不一致会推送失败
 
 3.注意 带宽/流量 相关指标的单位, 带宽值我们对外的统计界面会自行转换为 bps, 这里只需上报 bit 值

```
#### 接口状态
> 开发中

#### 接口URL
> https://vda.cug.cdnnow.cn/client/log_push

#### 请求方式
> POST

#### Content-Type
> json

#### 请求Body参数
```javascript
{
  "tenantId": "cug", // 租户id
  "start_time": 1761818110530, // 时间戳
  "country": "cn", // 国家
  "region": "mainland_china", // 地区
  "domain": "example.com", // 域名
  "bw": 10240000, // 带宽
  "flux": 20480000, // 流量
  "bs_bw": 512, // 回源带宽
  "bs_flux": 1024, // 回源流量
  "req_num": 1000, // 请求总数
  "hit_num": 800, // 命中请求数
  "bs_num": 200, // 回源请求数
  "bs_fail_num": 10, // 回源失败请求数
  "hit_flux": 1600, // 命中流量
  "http_code_2xx": 750, // HTTP 2XX 状态码数量
  "http_code_3xx": 50, // HTTP 3XX 状态码数量
  "http_code_4xx": 150, // HTTP 4XX 状态码数量
  "http_code_5xx": 50, // HTTP 5XX 状态码数量
  "bs_http_code_2xx": 180, // 回源 HTTP 2XX 状态码数量
  "bs_http_code_3xx": 10, // 回源 HTTP 3XX 状态码数量
  "bs_http_code_4xx": 8, // 回源 HTTP 4XX 状态码数量
  "bs_http_code_5xx": 2, // 回源 HTTP 5XX 状态码数量
  "interval": 300 // 时间粒度  300(5分钟) / 3600(1小时) / 86400(1天) / 2592000(1月)
}
```
参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述
--- | --- | --- | --- | ---
tenantId | cug | String | 是 | 租户id
start_time | 1741169158000 | Integer | 是 | 时间戳
country | cn | String | 是 | 国家, ISO 3166-1 alpha-2（二位字母代码) 全部小写
region | mainland_china | String | 是 | 地区, 自主映射到 {"mainland_china", "asia", "europe", "africa", "north_america", "south_america", "oceania", "antarctica", "other"} 中的一个
domain | example.com | String | 是 | 域名
bw | 1024 | Integer | 是 | 带宽 单位 bit, 1000 换算

租户界面给到的是平均带宽 bps, 即 (总量/时间间隔), 时间间隔单位是 s
flux | 2048 | Integer | 是 | 流量 单位 Byte, 1024 换算
bs_bw | 512 | Integer | 是 | 回源带宽 单位 单位 bit, 1000 换算
